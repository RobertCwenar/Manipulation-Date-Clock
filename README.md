# Manipulation-Date-Clock

## Opis projektu

Projekt służy do **manipulacji datą i czasem** w prosty i wygodny sposób. Umożliwia **dodawanie oraz odejmowanie dat i godzin**, niezależnie od długości okresu (dni, miesięcy, lat, godzin itp.), eliminując typowe problemy związane z ręcznymi obliczeniami czasu.

Dzięki zastosowaniu **Thinkera** możliwe jest **wyświetlanie kalendarza w zewnętrznej aplikacji desktopowej**, co zwiększa czytelność i komfort pracy z danymi czasowymi.

Celem projektu jest **usprawnienie pracy z datą i czasem**, szczególnie w sytuacjach wymagających precyzyjnych obliczeń, planowania lub analizy przedziałów czasowych, bez konieczności martwienia się o różną długość miesięcy czy lat przestępnych.

Projekt może być wykorzystany zarówno jako **samodzielne narzędzie**, jak i **moduł wspierający inne aplikacje**.
